# GCP Workload Security Agent

The GCP Workload Security Agent is a sub-agent of the `cloud_compliance_agent` responsible for assessing the security of workloads running in Google Cloud Platform (GCP).

## 🚀 Features

- **Workload Inventory:** Discovers and lists various types of workloads, including GCE instances, GKE clusters, and Cloud Run services.
- **Vulnerability Scanning:** Scans container images and VM instances for known vulnerabilities.
- **Runtime Security:** (Future) Monitors workloads in real-time for suspicious activity.
